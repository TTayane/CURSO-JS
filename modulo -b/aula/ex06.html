Operadores aritiméticos

  • + -> adição (ex: 5 +2 = 7)
  • - -> subtração (ex: 5 - 2 = 3)
  • * -> multiplicação (Ex: 5 *2= 10)
  • / -> divisão (ex: 5 /2= 2,5)
  • % -> resto da divisão por 2 (ex: 5 % 2 = 1)
  • **-> raiz (ex: 5**2=25)

nos operadores, aquele que vc quiser que ter o resultado primeiro precisa está entre parentese. ex: (5+3)/2= 4 --> primeiro soma depois divide. se não estivesse entre parentese o resultado seria: 5+3/2= 6.5, pois a preferência inicial será da divisão

Ordem de precedência dos operadores

  • 1° -> ()
  • 2° -> **
  • 3° -> * - / - %
  • 4° -> + -

auto-atribuição

var n=3
n= n+ 3 --> resultado será 7
n= n * 2 --> o resultado será 14

simplificando a varíavel

n=3
n+= 3 (7)
n*= 2 (14) ...

variavel de incremento

n=3
n++ (4)
n-- (3)

Operadores relacionais

  • 5 >2 = true
  • 7 < 2 = 4 = false
  • 8 >=8 = true
  • 9 <= 7 = false
  • 5 ==5 = true
  • != 4 = false

O símbolo de igual ==, não vai fazer distinsão entre formas, ex: 5 é igual a '5' apesar deste ultimo ser uma string, porém ele não será igual como 5==='5', a grandeza é a mesmo mas um pe numero e o outro string
5 ==5 -> true
5=="5" -> true
5 === 5 -> true
5 === '5' -> false

Operadores Lógicos

  • ! -> negação (true -> false / false-> true)
  • && -> conjunção
    true && true> true /
    true && false -> false /
    false && true -> false/
    false && false -> false
  • || -> disjunção
    true || true -> true/
    true || false -> true/
    false || true -> true/
    false || false -> false
